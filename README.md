# تجربة معرض غامرة (Immersive Expo)

موقع تفاعلي على غرار [jenamdvacetpet.cz](https://jenamdvacetpet.cz) — لكن لمركز معارض، حيث كل **مشروع/جناح** نقطة مرقّمة داخل القاعة تفتح قصة بشرائح.

## التشغيل
افتح `index.html` في المتصفح — لا يحتاج بناء. (يُفضّل عبر خادم محلي بسيط حتى تظهر الصور: `python3 -m http.server`).

## الأجزاء المرقّمة (للتعديل السهل)

| الرقم | الجزء | المعرّف |
|------|------|---------|
| 1 | شاشة التحميل + العدّاد | `section-1-loading` |
| 2 | المقدمة (كلمات تظهر تِباعًا) | `section-2-intro` |
| 3 | الشريط العلوي (شعار/صوت/لغة) | `section-3-chrome` |
| 4 | خريطة القاعة + النقاط المرقّمة | `section-4-map` |
| 5 | محطة المشروع (عنوان + رقم عملاق) | `section-5-station` |
| 6 | لوحة القصة (صورة + نص) | `section-6-story` |
| 7 | أدوات الترقيم/التنقّل | `section-7-nav` |

قل «عدّل القسم X» وسأعرف مكانه فورًا.

## ★ كيف تضيف مشاريعك/تصاميمك
كل المحتوى في قائمة واحدة أعلى `script.js` اسمها `STATIONS`:

```js
{
  num: '01',                    // الرقم على الخريطة والمحطة
  title: 'اسم المشروع',
  x: '28%', y: '52%',           // موضع النقطة في القاعة (x من اليمين)
  booth: 'assets/booth-01.jpg', // صورة الجناح (اختياري)
  slides: [                     // شرائح القصة
    { img:'assets/p1.jpg', caption:'تعليق الصورة', heading:'العنوان', text:'النص...' }
  ]
}
```

- ضع صورك في مجلد `assets/` ثم اكتب مسارها.
- لفتح نقطة مقفلة: احذف `locked:true` وأضف لها `slides`.
- لتغيير كلمات المقدمة: عدّل `INTRO_WORDS`.

## ملاحظة عن خلفية القاعة
القاعة حاليًا محاكاة بالـ CSS (أرضية بمنظور + إضاءة دافئة) كعنصر مؤقّت قابل للاستبدال.
عند تزويدي بتصميم مركز المعارض، أضعه خلفيةً عبر `.venue-bg { background-image:url('assets/venue.jpg') }`.

## المؤثّرات
عدّاد تحميل، تدرّج متحرك، ظهور كلمات تِباعًا، نقاط نابضة، تكبير سينمائي (Ken Burns)، انتقالات سلسة بين الشاشات، تنقّل بالأسهم ولوحة المفاتيح (Esc للرجوع).
