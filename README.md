# موقع نظام تصميم الذكرى السنوية

موقع مستقل (HTML/CSS/JS فقط) مبني بالكامل على نظام التصميم مع كل المؤثّرات.

## كيف تشغّله
افتح `index.html` مباشرة في المتصفح — لا يحتاج أي بناء أو تثبيت.

## الأجزاء المرقّمة (للتعديل لاحقًا بسهولة)

كل جزء له رقم ومعرّف (`id`) ثابت في `index.html`، ونفس الترقيم مكتوب في `styles.css`:

| الرقم | الجزء | المعرّف (id) |
|------|------|-------------|
| 1 | شريط التنقل العلوي | `section-1-nav` |
| 2 | الواجهة الرئيسية (Hero + التدرّج) | `section-2-hero` |
| 3 | الأرقام (عدّادات متحركة) | `section-3-stats` |
| 4 | المميزات (بطاقات) | `section-4-features` |
| 5 | معرض العناصر (أزرار/شارات/روابط) | `section-5-showcase` |
| 6 | نموذج التواصل | `section-6-form` |
| 7 | التذييل | `section-7-footer` |

عند طلب أي تعديل، قل رقم الجزء فقط (مثلاً: «عدّل القسم 4»).

## المؤثّرات المضمّنة
- تدرّج متحرك في الواجهة والتذييل + أشكال عائمة.
- ظهور تدريجي للعناصر عند التمرير (Scroll Reveal).
- عدّادات أرقام متحركة.
- تأثير تموّج (Ripple) عند الضغط على الأزرار.
- شريط تنقل يتغيّر عند التمرير + تمييز القسم النشِط + قائمة جوّال.
- حالات تفاعل كاملة للأزرار والروابط والحقول، وتحقّق من النموذج.

## الملفات
- `index.html` — البنية والمحتوى (الأقسام المرقّمة).
- `styles.css` — كل التنسيقات ومتغيّرات نظام التصميم.
- `script.js` — كل المؤثّرات والتفاعلات.
